Avatar

The Avatar component is used to represent a user, and displays the profile picture, initials or fallback icon.

Examples

The Avatar component comes in 6 sizes. Starting from 2xl, xl, lg, md, sm and xs. There are different shapes as well to give structures like circle, square.

Badges

Documentation and examples for badges, our small count and labeling component.

Examples

Badges scale to match the size of the immediate parent element by using relative font sizing and em units. Examples of badges containing text, using primary and secondary colors. The badge is applied to its children.
